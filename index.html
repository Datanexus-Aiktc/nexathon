What is Nexathon?

Nexathon is a 24-hour intensive data science and machine learning competition where innovators, developers, and data enthusiasts collaborate to solve challenging real-world problems. It's a platform to learn, build, and showcase your skills in a competitive yet supportive environment.

  • Focus: Data Science, Machine Learning, and AI.
  • Goal: Hack for impact by creating data-driven solutions.
  • Datasets: Work with unique, large-scale datasets provided by our partners.

Submission Guidelines

  1. Team size must be between 2 and 4 members.
  2. Submissions must include source code, a detailed README, and a short presentation video.
  3. The primary submission format should be a link to a public Git repository (e.g., GitHub, GitLab).
  4. Models will be evaluated based on accuracy, innovation, scalability, and presentation clarity.

Rules of Nexathon

All participants must adhere to our Code of Conduct to ensure a safe and respectful environment. All intellectual property developed during the hackathon remains with the participants. Plagiarism is strictly forbidden and will result in immediate disqualification. All work submitted must be created during the 24-hour event.
